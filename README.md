# VBA Reconciliation Tool

## 📌 Overview

This project is an Excel VBA automation tool developed to simplify the reconciliation process between two datasets. It compares records, identifies matches and mismatches, and reduces the time required for manual reconciliation.

---

## ✨ Features

- Automated data reconciliation
- Match and compare records across datasets
- Identify unmatched entries
- Reduce manual effort
- Improve accuracy and consistency

---

## 🛠 Technologies Used

- Microsoft Excel
- VBA (Visual Basic for Applications)

---

## 🚀 How to Use

1. Open the Excel workbook.
2. Load the required datasets.
3. Run the VBA macro.
4. Review the reconciliation results.

---

## 📈 Outcome

This project demonstrates practical Excel automation using VBA to streamline repetitive reconciliation tasks commonly performed in finance and accounting workflows.
